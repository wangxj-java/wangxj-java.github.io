本章知识点

选择题

  1. 中国封建社会的主要矛盾是地主阶级和农民阶级的矛盾
  2. 半封建半殖民的中国最重要的矛盾是帝国主义与中华民族的矛盾
  3. 帝国主义对中国最首要的侵略是军事侵略
  4. 1840年第一次鸦片战争,中英签订不平等条约《南京条约》又称《江宁条约》。 割香港给英国,开放广州、厦门、福州、宁波、上海等港口城市作通商口岸
  5. 第二次鸦片战争期间,英法联军火烧圆明园
  6. 中日甲午战争失败,中日签订《马关条约》割给日本台湾岛及附属岛屿和澎湖列岛
  7. 美国提出 “ 门户开放 ” 政策
  8. 1840年第一次鸦片战争是中国近现代史的开端，从此中国进入半封建半殖民地社会
  9. 半殖民地半封建社会的主要矛盾是国仇帝国主义与中国民族，其次家恨是封建主义与人民大众
  10. 1841年广州三元里斗争是中国近代史上中国人民第一次大规模反侵武装略斗争
  11. 中日甲午战争，邓世昌（致远舰），林永升牺牲在黄海战役，丁汝昌，刘步蟾牺牲于威海战争
  12. 中国近代史反侵略斗争失败的主要原因是社会制度的腐败，其次是经济技术的炉后
  13. 林则徐是近代史睁眼看世界第一人，组织编制《四洲志》
  14. 1843年，魏源编纂《海国图志》。提出“师夷长技以制夷”思想
  15. 1895年 严复翻译的《天演论》“物竞天择，适者生存

主观题

  • 资产阶级的两重性

    • 受帝，封压迫，一定条件下可以参加反帝反封的革命或保持中立
    • 因力量薄弱，与帝，封有着千丝万缕的关系，缺乏彻底的战斗精神

  • 工人阶级特点

    • 受帝，封，资三重压迫，革命性更强
    • 工人阶级人数虽少却易于集中组织成革命的力量
    • 他们大多来自破产的农民，与农民有天然联系，便于结成工农联盟
    • 最先进，最革命，最有力量的阶级

  • 半殖民地半封建社会的两大历史任务关系（独立【前提】 富强【目的】）

    • 历史任务：求得民族独立和人民解放和实现国家繁荣富强和人类共同富裕
    • 关系：既相互区别，又相互联系。区别：前者解决生产关系问题，后者解决生产力。完成第一大任务是为了第二大任务的完成创造条件

  • 半殖民地半封建社会特点：

    • 资帝支配：资本帝国主义不但逐步操纵了中国的财政和经济命脉，而且逐步控制了中国的政治，日益成为支配中国的决定性力量

    • 帝封勾结：中国的封建势力同外国侵略势力相勾结成为奴役中国人民的社会基础和统治支柱

    • 经济破坏：自然经济遭到破坏，但是封建剥削制度的根据地主的土地所有制依然存在

    • 资本发展：中国资本主义有所发展，在政治和文化生活上起到了一定的作用 但是发展不平衡，

    • 人民贫困: 人民毫无政治权利的生活极度贫困化

选择题

  1. 太平天国时间：（洪秀全）

    1. 1851年1月 金田起义

    3. 1856年9月 天京事变，由盛转衰

  2. 《天朝田亩制度》：最能体现太平天国的社会理想个这次农民战争特点的纲领性文件，是一个以解决土地问题为中心的比较完整的社会改革方案

  3. 《资政新篇》：洪仁玕提出，是第一个带有鲜明的资本主义色彩的改革方案与建设方案

  5. 洋务运动的指导思想 “中学为体，西学为用”，最早做出比较完整表述的是冯桂芬

  6. 洋务运动的领导是亦䜣

  7. 洋务运动的办事机构是 1861年成立的综合处理洋务的中央机关总理各国事务衙门，江南制造总局是第一个规模较大的近代军事工业的兵工厂。北洋水师是清政府的主力海军，北洋水师全军覆没标志着洋务运动的失败

  8. 著书立说

    1. 康有为：《新学伪经考》、《日本变政考》、《孔子改制考》和《人类公理》(三考一理);
    2. 梁启超：《变法通义》《时务报》;
    3. 谭嗣同《仁学》;
    4. 严复翻译《天演论》。

  9. 洋务派官僚，湖广总督张之洞写下了《劝学篇》，书中宣扬“中学为体，西学为用”的思想

  10. 维新派和守旧派的论战实质上是资产阶级思想与封建主义思想在中国近代史上第一次正面交锋

主观题

  • 太平天国历史意义

    • 沉重打击了封建统治阶级，撼动了清政府的通知根据
    • 中国旧时代农民起义的最高峰，《天朝田亩制度》《资政新篇》。
    • 批判儒家思想为代表的封建统治思想
    • 打击外国势力，维护中国主权
    • 掀起亚洲反殖民的大潮

  • 太平天国失败的原因

    • 领导：没有先进阶级的领导
    • 思想：没有科学理论的指导
    • 软弱：把信奉上帝的西方人称为“洋兄弟”

  • 洋务运动在历史上的作用

    • 工业：客观上促进工业和民族资本主义的发展
    • 教育：成为中国近代教育的开端
    • 文化：传播新知识，打开了人民的视界
    • 社会：引起社会风气和价值观念的变化

  • 洋务运动失败的原因（三性一阻）

    • 洋务运动具有封建性
    • 洋务运动对西方列强具有依赖性
    • 洋务企业的管理具有腐朽性
    • 顽固派的阻挠

  • 维新派与守旧派论战（主要一下三个问题）

    • 要不要变法
    • 要不要兴民权，设议院，实行君主立宪制
    • 要不要费八股，该科举个兴学堂

  • 戊戌变法失败的原因

    • 主要原因：以慈禧太后为首的强大的旧势力的反对
    • 自身原因：
      • 不敢否定封建主义
        • 政治：不敢否定封建君主专制制度
        • 经济：不敢否定封建土地所有制
        • 文化：不敢否定封建专制主义文化，托古改制
      • 对帝国主义抱有幻想
      • 脱离人民群众

  • 戊戌维新运动的意义及教训

    • 意义

      • 是一次爱国救亡运动
      • 是一场资产阶级性质的政治改革运动
      • 是一场思想启蒙
      • 在改革社会风气方面又不可低估的意义

    • 教训

      • 戊戌维新是中国民族资产阶级登上政治舞台的第一次表演

      • 戊戌维新以悲剧性的失败而告终暴露出这个阶级的软弱性

      • 说明在半殖民半封建社会，企图通过统治者自上而下的改良道路是行不通的

      • 要想实现国家的独立，富强，民主。必须采用强硬的革命手段

主观题

  • 武昌起义暴露的问题：

    • 资产阶级革命派的软弱性与妥协性：

      • 主动把权利让给立宪派或旧官僚，旧军官

      • 一些省份的旧官僚和立宪派改头换面的维持旧政权

      • 掌权的革命党人蜕变为新军阀，新官僚

        说明：革命虽然发展很快，但根据并不牢固，内外都潜伏着深刻的危机

  • 民族，民权，民生三大主义 简称三民主义

内容 措施 缺陷
民族主义 驱除鞑虏，恢复中华 以革命手段推翻清王朝 反对阻碍革命的满族人，但是没有明确反帝
民权主义 创立民国 建立资产阶级民主共和国 没有明确劳动人民的地位
民生主义 平均土地 核定地价，按价征税，涨价归公 没有触及到封建土地专有制

  • 中华民国临时政府

    • 性质：是一个资产阶级共和国性质的革命政权
      • 资产阶级：资产阶级革命派控制着这个政权
      • 共和：集中体现了中国民族资产阶级的利益和愿望，也一定程度上符合广大中国人民的利益

  • 辛亥革命的历史意义和局限性

    • 历史意义

      • 思：传播了民主共和的理念，推动了思想解放
      • 变：促使社会经济，思想习惯和社会风气等方面发生新的变化
      • 清：它推翻了清王朝在中国的统治，沉重打击了中外反动势力在中国的统治
      • 帝：打击帝国主义在华势力，为亚洲各国人民革命斗争提供范例，推动亚洲各国民族解放运动
      • 国：宣告了封建君主专有制度的结束和民国的建立

    • 局限性

      • 资产阶级革命派自身的软弱性：
        1. 纲：没有提出彻底的反对帝国主义和反对封建主义革命纲领
        2. 人：没有充分发动和依靠人民
        3. 党：没有建立坚强的革命政党，发挥团结一起革命的力量的强有力的核心作用

    • 新文化运动的历史意义

      • 资产阶级民主主义新文化和同封建主义旧文化的斗争，沉重打击封建专有制主义文化
      • 它大力宣传了民主与科学，将人们从封建专制的蒙昧中解放出来，开启了思想解放的高潮
      • 它为马克思主义传入准备了适宜的土壤，为中国新民主主义创造了思想文化上的条件

      中国旧民主主义革命终结的标志是（第二次护法运动的失败）

  • 新文化运动与五四运动

    • 局限性：
      • 脱离经济：简单的把资产阶级共和国方案失败的根本原因归之于的思想文化
      • 脱离群众：没有把运动普及到工农群众中去
      • 脱离实际：少数人在思想上方法上存在绝对肯定或否定的形式主义偏向

主观题

  • 新文化运动的历史意义

    1. 资产阶级民主主义新文化与封建主义旧文化的斗争，沉重打击了封建专制主义文化
    2. 它大力宣传了民主与科学，将人民从封建专制蒙昧中解放出来，开启了思想解放的潮流
    3. 它为马克思主义的传入准备了适宜的土壤，为中国新民主主义革命创造了思想文化上的条件

  • 五四运动的历史意义

    1. 青年学生是先锋，工人阶级第一次独立登录政治舞台，成为主力军
    2. 促进了马克思主义在中国的广泛传播
    3. 中国近代史上第一次彻底的反封建反帝的革命运动
    4. 是中国新民民主主义革命的伟大开端

  • 中国共产党成立的意义

    1. 领导：标志中国革命有坚强的领导核心
    2. 思想：中国革命从此有科学的指导思想
    3. 世界：沟通中国与世界革命的联系，是世界无产阶级革命运动的一部分

  • 北伐战争的胜利原因

    1. 国共合作的实现，革命统一战线的成立，共产党员和共青团员的先锋模范是重要原因
    2. 是在反帝，反对军阀的口号下进行争议的革命战争，得到人民群众的大力支持
    3. 得到苏联政府的援助，特别是派出的军事顾问帮助北伐军制定了正确的军事战略

  • 大革命的意义

    1. 沉重打击了帝国主义和封建主义的统治势力，实际上是迎接未来革命胜利的一次伟大的演习
    2. 扩大了中国共产党在中国人民中的政治影响，使党经受了一次大革命的“洗礼”

  • 中国民族资本主义经济特点（三小一轻一联系）

    1. 民族资本小：民族资本主义经济在国民经济中所占比很小
    2. 工业资本小：在民族资本主义中，工业资本所占的很小
    3. 规模小：技术设备落后，劳动生产率低，企业规模狭小
    4. 轻工业多：民族资本主义主要以纺织，食品工业为主的轻工业
    5. 封建联系：民族资本主义也与封建势力有着千丝万缕的联系

第一节 国民党在全国的统治和中间党派的政治主张

国民党统治建立：

  1. 宁汉合流：南京国民政府改组，武汉国民政府不再存在，张学良“东北易帜”，标志着国民党在全国范围内建立起一党专政和军事独裁统治

  2. 国民党代表大地主，买办性的大资产阶级利益的一党专政和军事独裁统治

  3. 蒋介石反动政权的经济基础是国家垄断资本，这种垄断活动首先和最主要是从金融业方面开始的

  4. 在国民党政府统治期间，有一些中间党派活跃在国共两党之间，政治主张上既不满意国民党
    的专制独裁，也不同意共产党的纲领路线。希望通过自己改良主义运动，找到适合中国发展
    资本主义的道路。

    邓演达领导中国国民党临时行动委员会（又称第三党）

    梁漱溟为首的乡村建设派

    黄炎培为首的中华职业教育社

第二节 中国共产党对革命新道路的艰苦探索

在第一次国共合作后期，蒋介石和汪精卫反动政变，捕杀共产党员。国民党的统治建立更是大肆围杀共产党员，这是都是共产党人意识到武装反抗斗争的重要性

  1. 1927年8月1日，周恩来，贺龙，叶挺，朱德，刘伯承等人，率领北伐军2万人在南昌起义

    南昌起义的意义：

    • 它打响了武装反抗国民党反动统治的第一枪
    • 它成为共产党独立领导的革命战争，创建人民军队和武装夺取政权的伟大开端
    • 它揭开了土地革命战争的序幕

  2. 1927.8.7 八七会议的主要内容是确定“土地革命和武装斗争”，毛泽东并提出“须知政权是由枪杆子中取得的”

    八七会议是大革命失败到土地革命战争兴起的一个历史转折点

  3. 1927.9.9 毛泽东领导秋收起义，目的是打下长沙， 结果失败：

    秋收起义的意义：

    • 公开打出了“工农革命军”的旗帜
    • 有数量众多的工农武装参加

  4. 三湾改编成为建设中国共产党领导的 新型人民军队 的重要开端

  5. 1927.10.7 毛泽东在江西宁冈县茅坪开始创建 “井冈山农村革命根据地”的斗争

    以毛泽东为代表的共产党人探索出一条适合中国革命的新道路：农村包围城市，武装夺取政权

  6. 第一次明确提出了 “工农武装割据” 的思想：《中国红色政权为什么能够存在？》《井冈山的战斗》

  7. 提出以乡村包围城市的思想：《星星之火，可以燎原》

  8. 坚持辩证唯物主义的思想：《反对本本主义》

  9. 中央革命根据地：1930.10到1931.7 红一军在毛泽东和朱德的领导下，连续粉碎三次围剿，开辟了中央革命根据地

  10. 土地革命：

    1. ”井冈山土地法“：是中国共产党历史上第一个土地法
    2. ”兴国土地法“：第二个土地法，将”没收一切土地“改为”没收一切公共土地及地主阶级的土地“，保护了中农的利益。
第三节 中国革命在探索中曲折前进

  1. 中华苏维埃第一次全国代表大会，毛泽东当选中央执行委员会主席

  2. 党内三次“左”倾错误

    1. 1927-1928年以瞿秋白为代表的左倾盲动错误。
    2. 1930 年以李立三为代表的左倾冒险主义
    3. 1931-1935年以王明为代表的左倾教条主义（首要）

  3. 1935.1遵义会议解决了当时具有决定意义的军事和组织问题，张闻天对党负总的责任，成立由毛泽东，周恩来，王稼祥组成的新的三人团，全权负责红军的军事行动

  4. 1935.10，中央红军到达陕北吴起镇，同十五军团会合

  5. 1936.10，红二，四（张国焘，徐向前）方面军先后同红一军在甘肃会宁，静宁将台堡会师，胜利结束长征

主观题

  • 中国民族资本主义的经济特点

    • 民族资本小：民族资本经济在国民经济中所占比小
    • 工业资本小：在民族资本中，工业资本所占比小
    • 规模小：技术设备落后，生产效率低，企业规模小
    • 轻工业为主：民族资产主义中主要以纺织和食品为首的轻工业，重工业少
    • 封建联系：民族资产阶级与封建主义有着千丝万缕的联系

  • 毛泽东在三湾改编的内容：

    • 将一个团缩编为一个师
    • 建立中国革命军的各级别共产党及组织，将党的支部建立在连上
    • 成立各级士兵委员会，部队内部实行民主管理

  • 科学回答中国的红色政权为什么能够存在？

    • 根本原因：被几个帝国主义间接的统治并且在经济上发展极不平衡的半殖民半封建的大国
    • 客观影响：国民革命的影响
    • 主观影响：相当力量正式红军的存在

  • 遵义会议的历史意义：

    三挽一转

    • 挽救了中国共产党
    • 挽救了中国革命
    • 挽救了中国工农红军
    • 确立以毛泽东为代表的中央的领导，成为中国共产党在历史上一个生死攸关的转折点

    政治成熟

    • 标志着中国共产党政治走向成熟的阶段，证明中共具有自我净化和自我发展的能力

  • 长征的胜利及其伟大意义（转西北，锤精火）

    • 转：长征粉碎了国民党”围剿“红军，消灭革命力量的企图，是中国革命转危为安的关键
    • 西北：通过长征，把中国革命大本营放在了西本，为迎接中国人民抗日救亡的新高潮准备条件
    • 锤：长征保存并锤炼了中国革命的骨干力量
    • 精：长征铸就了伟大的长征精神
    • 火：长征广泛撒播了革命的火种
